
# Codigos de Pruba Actividad I y II

La presente carpeta contiene los documentos de prueba correspondientes a los programas desarrollados en las actividades anteriores. En ella se incluyen las evidencias que demuestran el funcionamiento de los códigos implementados, tanto para la gestión de clientes del consultorio odontológico como para la aplicación de estructuras de datos tipo pila y cola.

Cada documento presenta ejemplos de ejecución, resultados obtenidos y validación de los requerimientos solicitados, permitiendo verificar el correcto funcionamiento de las soluciones propuestas.

Estos archivos se adjuntan como soporte del trabajo realizado y como evidencia del cumplimiento de los criterios establecidos en la actividad.
